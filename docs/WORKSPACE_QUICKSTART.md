## Frequently Asked Questions

**Q: Where is my data stored?**
A: In Firebase Firestore (Google's secure cloud database)

**Q: Is my data safe?**
A: Yes! Only you can access your data. It's encrypted in transit.

**Q: Can I use on multiple devices?**
A: Yes! Data syncs automatically across devices.

**Q: What if I lose internet?**
A: App still works, syncs when back online.

**Q: Can I delete my data?**
A: Yes! Click "Reset" to clear mood history, or delete account.

**Q: How long is data kept?**
A: Until you delete it. No expiration.

## Troubleshooting

### Can't sign in?
1. Check internet connection
2. Try incognito/private browsing
3. Clear cookies and cache
4. Check Firebase is configured correctly

### Data not saving?
1. Check you're signed in
2. Look for error messages (F12)
3. Try refreshing page
4. Check Firestore Rules are set

### Slow performance?
1. Check internet speed
2. Refresh page
3. Clear browser cache
4. Check Firebase quota

### Missing data?
1. Check you're logged in as correct user
2. Refresh page
3. Check Firestore in Firebase Console
4. Contact support with screenshot
