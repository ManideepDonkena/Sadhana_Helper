## Troubleshooting Guide

### Firebase Not Working
```
1. Check Firebase config in workspace.js
2. Verify Firestore exists in Firebase Console
3. Check Firestore Rules are published
4. Clear browser cookies and cache
5. Check browser console (F12) for errors
```

### Data Not Syncing
```
1. Verify you're signed in
2. Check Firestore Rules allow read/write
3. Look for network errors
4. Verify user UID matches
5. Try signing out and back in
```

### Slow Performance
```
1. Check internet connection
2. Refresh the page
3. Clear browser cache
4. Check Firebase quota usage
5. Try different browser
```
